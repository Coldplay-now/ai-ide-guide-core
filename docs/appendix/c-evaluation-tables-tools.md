## C.2 ROI计算工具和模板


### C.2.1 ROI计算公式


**基础ROI计算**


```markdown
ROI = (收益 - 投资成本) / 投资成本 × 100%

其中：
- 收益 = 效率提升带来的成本节省
- 投资成本 = 工具费用 + 培训费用 + 实施费用

```

**详细ROI计算模板**


